# Employee Burnout Prediction Using Machine Learning

## Overview
This project predicts employee burnout levels using machine learning techniques based on employee work-related factors such as mental fatigue score, resource allocation, work-from-home setup, and designation.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Building
- Model Evaluation

## Dataset
Employee Burnout Analysis Dataset

Features used:
- Gender
- Company Type
- WFH Setup Available
- Designation
- Resource Allocation
- Mental Fatigue Score
- Burn Rate (Target Variable)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- OpenPyXL

## Machine Learning Models Used
1. Linear Regression
2. Random Forest Regressor

## Model Performance

Linear Regression R² Score: 0.92

Random Forest Regressor R² Score: 0.91

## Conclusion
The project successfully predicts employee burnout levels using machine learning models.

Among the tested models, Linear Regression performed slightly better with an R² score of 0.92.

The analysis showed that:
- Mental fatigue strongly affects burnout
- Higher resource allocation increases burnout risk
- Work-from-home setup impacts employee burnout
